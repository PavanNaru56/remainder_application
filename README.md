REMAINDER APPLICATION:
~~~~~~~~~~~~~~~~~~~~~


Steps to execute:

|-> Copy the link from GitHub and clone the GitHub repository using your default editor or Visual Studio Code.

|-> To install the required node_modules, use the command: npm install or npm install --force.

|-> Set the email address in the 'to:' field to receive notifications via email at Remainders/set.js.

|-> Use the command node index.js to run the application.

|-> Set the reminder for the required time.

|-> Finally, you will be notified as described in the assignment.
